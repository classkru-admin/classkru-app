# Web Styling — ตกแต่งหน้าเว็บ + คิดสีอัตโนมัติ

สกิลนี้ทำหน้าที่ "นักออกแบบสี" เลือกและเติมสีให้หน้าเว็บออกมา **โมเดิร์นแต่สะอาดตา** แนวทางเริ่มต้น: **เขียวพาสเทล สไตล์ Soft/Friendly** เข้าชุดกับ ClassKru

> สกิลนี้คุม "สีและความสวย" — ส่วนโครง layout/component ใช้ [web-design-classkru]

## ปรัชญา Soft / Friendly

- พื้นหลังสว่าง โปร่ง ไม่ทึบ — เน้น whitespace ให้หายใจ
- สี accent ใช้แค่จุดสำคัญ (ปุ่มหลัก, สถานะ active) ไม่ระบายเต็มหน้า
- ขอบมน เงานุ่ม โทนพาสเทลอบอุ่น ไม่ฉูดฉาด
- ความคมชัดของตัวอักษรต้องมาก่อนความสวยเสมอ (อ่านง่าย)

## กระบวนการคิดสี (ทำตามขั้นนี้ทุกครั้ง)

**1. เลือกสี accent หลัก**
ค่าเริ่มต้น: เขียวพาสเทล `#1d9e75` (เข้าชุด ClassKru) ถ้างานอื่นให้เลือกสีที่สื่ออารมณ์ของเนื้อหา แต่คุมความสด (saturation) ให้พอดี ไม่จัดจนแสบตา

**2. สร้างเฉดของ accent (อ่อน→เข้ม)**
ทุก accent ต้องมีอย่างน้อย 4 เฉด:
- `--primary` สีหลัก (ปุ่ม, active)
- `--pk` เข้มกว่า ~12% (hover, ตัวอักษรบนพื้นอ่อน)
- `--pbg` พื้นอ่อนมาก (พื้น state active, แท็ก) — เฉดเดียวกันแต่ lightness สูง
- `--pbd` เส้นขอบ/เฉดกลางอ่อน

ตัวอย่างชุดเขียวพาสเทล (default):
```css
--primary:#1d9e75; --pk:#0f6e56; --pbg:#f0fdf4; --pbd:#86efac;
```

**3. วาง neutral ที่ "ไม่เทาตาย"**
อย่าใช้เทากลางๆ ล้วน ให้ผสมโทนของ accent ลงไปนิดเพื่อความกลมกลืน:
```css
--bg:#f5f7fa; --card:#fff; --text:#0f172a; --muted:#64748b; --border:#e8edf2;
```

**4. เงา + ทรานสิชัน นุ่ม**
```css
--sh:0 1px 4px rgba(0,0,0,.06); --sh-md:0 4px 16px rgba(0,0,0,.10);
--tr:all .18s cubic-bezier(.4,0,.2,1);
```

**5. เช็ก contrast — ขั้นบังคับ**
- ตัวอักษรเนื้อหาบนพื้น ต้องผ่าน WCAG AA (อัตราส่วน ≥ 4.5:1)
- ข้อความบนปุ่มสี accent ต้องอ่านชัด (ปกติใช้ตัวอักษรขาว `#fff` บน `--primary`)
- อย่าวางตัวอักษรสีพาสเทลอ่อนบนพื้นขาว (จาง อ่านไม่ออก) — เฉดอ่อนใช้เป็น "พื้น" เท่านั้น ไม่ใช่ตัวอักษร

ถ้าไม่แน่ใจ ให้คำนวณ contrast ratio จริง (เขียนสคริปต์สั้นๆ ก็ได้) ก่อนยืนยันสี

## วิธีเติมสีที่ "ขาด"

เมื่อหน้าเว็บมีแค่สีหลัก แล้วต้องการสีเพิ่ม (success/warning/danger/info) ให้คิดเป็นชุดเดียวกัน คุม saturation/lightness ใกล้กัน เพื่อให้อยู่ตระกูลพาสเทลเหมือนกัน:
```css
--success:#1d9e75; /* เขียว = สีหลักอยู่แล้ว */
--warning:#f59e0b; --warning-bg:#fffbeb;
--danger:#ef4444;  --danger-bg:#fef2f2;
--info:#3b82f6;    --info-bg:#eff6ff;
```
ทุกสีสถานะมี `-bg` เฉดอ่อนคู่กันเสมอ เพื่อใช้เป็นพื้น chip/alert แบบ Soft

## Workflow

1. ถ้ามีหน้าเดิม อ่าน `:root` เดิมก่อน ต่อยอดจากของจริง ไม่รื้อทั้งหมด
2. คิดสีตามขั้น 1–5 ข้างบน
3. เช็ก contrast ทุกคู่ตัวอักษร/พื้น
4. ใช้สีผ่าน CSS variable เท่านั้น — ห้าม hardcode hex นอก `:root`
5. แสดงตัวอย่าง palette (swatch) ให้ผู้ใช้ดูก่อนนำไปใช้จริงถ้าเป็นการเปลี่ยนโทนใหญ่

## อย่าทำ

- อย่าใช้ accent มากกว่า 1 ตระกูลสีในหน้าเดียว (ยกเว้นสีสถานะ)
- อย่าระบายสี accent เต็มพื้นที่ใหญ่ — Soft style เน้นพื้นสว่าง
- อย่าใช้พาสเทลกับตัวอักษรเนื้อหา (contrast ไม่พอ)
- อย่าทิ้ง contrast เพื่อความสวย
