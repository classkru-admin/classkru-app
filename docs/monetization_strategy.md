# 💰 Monetization Strategy — Thai Curriculum Copilot
> อัปเดต: มิถุนายน 2026

---

## โมเดล: Freemium → Paid per Indicator

### หลักการ
ครูเลือกตัวชี้วัดได้เอง → ใช้ฟรี 3 ตัว → ตัวที่ 4 เป็นต้นไปต้องจ่าย

---

## Freemium Rules

| | รายละเอียด |
|--|------------|
| **จำนวนฟรี** | 3 ตัวชี้วัด (เลือกได้เอง ไม่จำกัดว่าต้องเป็นตัวไหน) |
| **ครูเลือกเองได้** | ✅ ใช่ — ครูเลือก 3 ตัวชี้วัดที่ต้องการเอง |
| **Content ฟรี** | ครบ: แผนการสอน + ใบงาน + Quiz + เฉลย + Rubric |
| **Content จ่าย** | ครบเหมือนกันทุกตัว ไม่ลดคุณภาพ |
| **หมดอายุ** | ❌ ไม่หมดอายุ — ฟรีตลอดชีพ 3 ตัว |
| **บัญชีต้องการ** | ✅ ต้องสมัคร (เพื่อ track จำนวนที่ใช้ไป) |

---

## ราคา (Paid)

### Phase 1 — ขายรายตัวชี้วัด
| Package | เนื้อหา | ราคา |
|---------|---------|------|
| Single Indicator | แผนการสอน + ใบงาน + Quiz + เฉลย + Rubric (1 ตัวชี้วัด) | **79 บาท** |
| Bundle 3 Indicators | 3 ตัวชี้วัด (เลือกเองได้) | **199 บาท** (~66/ตัว) |
| Full Unit | ทุกตัวชี้วัดใน 1 หน่วย (เช่น ระบบนิเวศทั้ง 6 ตัว) | **390 บาท** (~65/ตัว) |

### Phase 2 — Subscription (เมื่อ validate แล้ว)
| Plan | เนื้อหา | ราคา |
|------|---------|------|
| Monthly | ไม่จำกัดตัวชี้วัด วิทย์ ม.3 | **99 บาท/เดือน** |
| Semester | 6 เดือน | **490 บาท** (~82/เดือน) |
| Annual | 1 ปี | **790 บาท** (~66/เดือน) |

## UX Flow

```
ครูเข้าระบบ
    ↓
เลือก: ม.3 → วิทยาศาสตร์ → หน่วย → ตัวชี้วัด
    ↓
[ฟรี] ยังใช้ไม่ถึง 3 ตัว
    → Generate ทันที ✅
    
[ต้องจ่าย] ใช้ครบ 3 ตัวแล้ว
    → แสดง: "คุณใช้สิทธิ์ฟรี 3 ตัวครบแล้ว"
    → ปุ่ม: "ปลดล็อคตัวชี้วัดนี้ 79 บาท"
    → ปุ่ม: "ซื้อ Bundle 3 ตัว 199 บาท"
    → ปุ่ม: "ซื้อทั้งหน่วย 390 บาท"
```

---

## เหตุผลที่โมเดลนี้ได้ผล

1. **ครูได้ลองของจริงก่อน** — ไม่ใช่แค่ตัวอย่าง แต่ได้ของจริงครบชุด
2. **ครูเลือกเองได้** — เลือกตัวชี้วัดที่สอนจริง ไม่ใช่ที่เราบังคับ
3. **3 ตัวพอเห็นคุณค่า** — ใช้ฟรีแล้วประหยัดเวลาได้จริง → อยากซื้อต่อ
4. **ราคา 79 บาท ต่ำกว่าจิตวิทยา 100** → ตัดสินใจง่าย
5. **ไม่หมดอายุ** — ลด anxiety ของครู ไม่รู้สึกถูกบังคับ

## KPI ที่ต้องวัด

| เดือน 1 | เป้าหมาย |
|---------|---------|
| ครูที่สมัครใช้ | 50+ คน |
| ครูที่ใช้ฟรีจนครบ 3 ตัว | 30+ คน |
| Conversion Rate (ฟรี→จ่าย) | >20% |
| รายได้ | 500–1,500 บาท |
| คำถามสำคัญที่ต้องรู้ | "ครูที่ใช้ฟรีหมด 3 ตัวแล้ว จ่ายต่อไหม?" |
