## สาระที่ 1 | วิทยาศาสตร์ชีวภาพ

### มาตรฐาน ว 1.1 — ระบบนิเวศ
> เข้าใจความหลากหลายของระบบนิเวศ ความสัมพันธ์ระหว่างสิ่งมีชีวิตกับสิ่งมีชีวิต และสิ่งไม่มีชีวิต การถ่ายทอดพลังงาน การเปลี่ยนแปลงแทนที่ในระบบนิเวศ รวมทั้งนำความรู้ไปใช้ประโยชน์

| รหัส | ตัวชี้วัด | ประเภท |
|------|-----------|--------|
| ว 1.1 ม.3/1 | อธิบายปฏิสัมพันธ์ขององค์ประกอบของระบบนิเวศที่ได้จากการสำรวจ | ระหว่างทาง |
| ว 1.1 ม.3/2 | อธิบายรูปแบบความสัมพันธ์ระหว่างสิ่งมีชีวิตกับสิ่งมีชีวิตรูปแบบต่าง ๆ ในแหล่งที่อยู่เดียวกันที่ได้จากการสำรวจ | ระหว่างทาง |
| ว 1.1 ม.3/3 | สร้างแบบจำลองในการอธิบายการถ่ายทอดพลังงานในสายใยอาหาร | ระหว่างทาง |
| ว 1.1 ม.3/4 | อธิบายความสัมพันธ์ของผู้ผลิต ผู้บริโภค และผู้ย่อยสลายสารอินทรีย์ในระบบนิเวศ | ระหว่างทาง |
| ว 1.1 ม.3/5 | อธิบายการสะสมสารพิษในสิ่งมีชีวิตในโซ่อาหาร | ระหว่างทาง |
| ว 1.1 ม.3/6 | ตระหนักถึงความสัมพันธ์ของสิ่งมีชีวิตและสิ่งแวดล้อมในระบบนิเวศ โดยไม่ทำลายสมดุลของระบบนิเวศ | ปลายทาง |

### มาตรฐาน ว 1.3 — พันธุกรรม ความหลากหลายทางชีวภาพ และวิวัฒนาการ
> เข้าใจกระบวนการและความสำคัญของการถ่ายทอดลักษณะทางพันธุกรรม สารพันธุกรรม การเปลี่ยนแปลงทางพันธุกรรมที่มีผลต่อสิ่งมีชีวิต ความหลากหลายทางชีวภาพและวิวัฒนาการของสิ่งมีชีวิต

| รหัส | ตัวชี้วัด | ประเภท |
|------|-----------|--------|
| ว 1.3 ม.3/1 | อธิบายความสัมพันธ์ระหว่างยีน ดีเอ็นเอ และโครโมโซม โดยใช้แบบจำลอง | ระหว่างทาง |
| ว 1.3 ม.3/2 | อธิบายการถ่ายทอดลักษณะทางพันธุกรรมจากการผสม โดยพิจารณาลักษณะเดียวที่แอลลีลเด่นข่มแอลลีลด้อยอย่างสมบูรณ์ | ระหว่างทาง |
| ว 1.3 ม.3/3 | อธิบายการเกิดจีโนไทป์และฟีโนไทป์ของลูก และคำนวณอัตราส่วนการเกิดจีโนไทป์และฟีโนไทป์ของรุ่นลูก | ระหว่างทาง |
| ว 1.3 ม.3/4 | อธิบายความแตกต่างของการแบ่งเซลล์แบบไมโทซิสและไมโอซิส | ระหว่างทาง |
| ว 1.3 ม.3/5 | บอกได้ว่าการเปลี่ยนแปลงของยีนหรือโครโมโซมอาจทำให้เกิดโรคทางพันธุกรรม พร้อมทั้งยกตัวอย่าง | ระหว่างทาง |
| ว 1.3 ม.3/6 | ตระหนักถึงประโยชน์ของความรู้เรื่องโรคทางพันธุกรรม โดยรู้ว่าก่อนแต่งงานควรปรึกษาแพทย์ | ปลายทาง |
| ว 1.3 ม.3/7 | อธิบายการใช้ประโยชน์จากสิ่งมีชีวิตดัดแปรพันธุกรรม (GMO) และผลกระทบที่อาจมีต่อมนุษย์และสิ่งแวดล้อม | ระหว่างทาง |
| ว 1.3 ม.3/8 | ตระหนักถึงประโยชน์และผลกระทบของสิ่งมีชีวิตดัดแปรพันธุกรรม โดยการเผยแพร่ความรู้ที่มีข้อมูลสนับสนุน | ปลายทาง |
| ว 1.3 ม.3/9 | เปรียบเทียบความหลากหลายทางชีวภาพในระดับชนิดสิ่งมีชีวิตในระบบนิเวศต่าง ๆ | ระหว่างทาง |
| ว 1.3 ม.3/10 | อธิบายความสำคัญของความหลากหลายทางชีวภาพที่มีต่อการรักษาสมดุลของระบบนิเวศและต่อมนุษย์ | ระหว่างทาง |
| ว 1.3 ม.3/11 | แสดงความตระหนักในคุณค่าและความสำคัญของความหลากหลายทางชีวภาพ โดยมีส่วนร่วมในการดูแลรักษา | ปลายทาง |

## สาระที่ 2 | วิทยาศาสตร์กายภาพ

### มาตรฐาน ว 2.1 — สสารและปฏิกิริยาเคมี
> เข้าใจสมบัติของสสาร องค์ประกอบของสสาร ความสัมพันธ์ระหว่างสมบัติของสสารกับโครงสร้างและแรงยึดเหนี่ยวระหว่างอนุภาค การเกิดสารละลาย และการเกิดปฏิกิริยาเคมี

| รหัส | ตัวชี้วัด | ประเภท |
|------|-----------|--------|
| ว 2.1 ม.3/1 | ระบุสมบัติทางกายภาพและการใช้ประโยชน์วัสดุประเภทพอลิเมอร์ เซรามิก และวัสดุผสม โดยใช้หลักฐานเชิงประจักษ์และสารสนเทศ | ระหว่างทาง |
| ว 2.1 ม.3/2 | ตระหนักถึงคุณค่าของการใช้วัสดุประเภทพอลิเมอร์ เซรามิก และวัสดุผสม โดยเสนอแนะแนวทางการใช้วัสดุอย่างประหยัดและคุ้มค่า | ปลายทาง |
| ว 2.1 ม.3/3 | อธิบายการเกิดปฏิกิริยาเคมี รวมถึงการจัดเรียงตัวใหม่ของอะตอมเมื่อเกิดปฏิกิริยาเคมี โดยใช้แบบจำลองและสมการข้อความ | ระหว่างทาง |
| ว 2.1 ม.3/4 | อธิบายกฎทรงมวล โดยใช้หลักฐานเชิงประจักษ์ | ระหว่างทาง |
| ว 2.1 ม.3/5 | วิเคราะห์ปฏิกิริยาดูดความร้อนและปฏิกิริยาคายความร้อน จากการเปลี่ยนแปลงพลังงานความร้อนของปฏิกิริยา | ระหว่างทาง |
| ว 2.1 ม.3/6 | อธิบายปฏิกิริยาการเกิดสนิมของเหล็ก ปฏิกิริยาของกรดกับโลหะ ปฏิกิริยาของกรดกับเบส ปฏิกิริยาของเบสกับโลหะ ปฏิกิริยาการเผาไหม้ การเกิดฝนกรด การสังเคราะห์ด้วยแสง | ระหว่างทาง |
| ว 2.1 ม.3/7 | ระบุประโยชน์และโทษของปฏิกิริยาเคมีที่มีต่อสิ่งมีชีวิตและสิ่งแวดล้อม และยกตัวอย่างวิธีการป้องกันและแก้ปัญหา | ระหว่างทาง |
| ว 2.1 ม.3/8 | ออกแบบวิธีแก้ปัญหาในชีวิตประจำวันโดยใช้ความรู้เกี่ยวกับปฏิกิริยาเคมี โดยบูรณาการวิทยาศาสตร์ คณิตศาสตร์ เทคโนโลยี และวิศวกรรมศาสตร์ | ปลายทาง |

### มาตรฐาน ว 2.3 — ไฟฟ้า คลื่น และแสง
> เข้าใจความหมายของพลังงาน การเปลี่ยนแปลงและการถ่ายโอนพลังงาน ปฏิสัมพันธ์ระหว่างสสารและพลังงาน ธรรมชาติของคลื่น ปรากฏการณ์ที่เกี่ยวข้องกับเสียง แสง และคลื่นแม่เหล็กไฟฟ้า

| รหัส | ตัวชี้วัด | ประเภท |
|------|-----------|--------|
| ว 2.3 ม.3/1 | วิเคราะห์ความสัมพันธ์ระหว่างความต่างศักย์ กระแสไฟฟ้า และความต้านทาน คำนวณโดยใช้ V = IR | ระหว่างทาง |
| ว 2.3 ม.3/2 | เขียนกราฟความสัมพันธ์ระหว่างกระแสไฟฟ้าและความต่างศักย์ไฟฟ้า | ระหว่างทาง |
| ว 2.3 ม.3/3 | ใช้โวลต์มิเตอร์ แอมมิเตอร์ในการวัดปริมาณทางไฟฟ้า | ระหว่างทาง |
| ว 2.3 ม.3/4 | วิเคราะห์ความต่างศักย์ไฟฟ้าและกระแสไฟฟ้าในวงจรไฟฟ้าเมื่อต่อตัวต้านทานหลายตัวแบบอนุกรมและแบบขนาน | ปลายทาง |
| ว 2.3 ม.3/5 | เขียนแผนภาพวงจรไฟฟ้า แสดงการต่อตัวต้านทานแบบอนุกรมและขนาน | ระหว่างทาง |
| ว 2.3 ม.3/6 | บรรยายการทำงานของชิ้นส่วนอิเล็กทรอนิกส์อย่างง่ายในวงจรจากข้อมูลที่รวบรวมได้ | ปลายทาง |
| ว 2.3 ม.3/7 | เขียนแผนภาพและต่อชิ้นส่วนอิเล็กทรอนิกส์อย่างง่ายในวงจรไฟฟ้า | ระหว่างทาง |
| ว 2.3 ม.3/8 | อธิบายและคำนวณพลังงานไฟฟ้าโดยใช้สมการ W = Pt รวมทั้งคำนวณค่าไฟฟ้าของเครื่องใช้ไฟฟ้าในบ้าน | ระหว่างทาง |
| ว 2.3 ม.3/9 | ตระหนักในคุณค่าของการเลือกใช้เครื่องใช้ไฟฟ้า โดยนำเสนอวิธีการใช้เครื่องใช้ไฟฟ้าอย่างประหยัดและปลอดภัย | ปลายทาง |
| ว 2.3 ม.3/10 | สร้างแบบจำลองที่อธิบายการเกิดคลื่นและบรรยายส่วนประกอบของคลื่น | ระหว่างทาง |
| ว 2.3 ม.3/11 | อธิบายคลื่นแม่เหล็กไฟฟ้าและสเปกตรัมคลื่นแม่เหล็กไฟฟ้าจากข้อมูลที่รวบรวมได้ | ระหว่างทาง |
| ว 2.3 ม.3/12 | ตระหนักถึงประโยชน์และอันตรายจากคลื่นแม่เหล็กไฟฟ้า โดยนำเสนอการใช้ประโยชน์และอันตรายในชีวิตประจำวัน | ปลายทาง |
| ว 2.3 ม.3/13 | ออกแบบการทดลองและดำเนินการทดลองด้วยวิธีที่เหมาะสมในการอธิบายกฎการสะท้อนของแสง | ระหว่างทาง |
| ว 2.3 ม.3/14 | เขียนแผนภาพการเคลื่อนที่ของแสงแสดงการเกิดภาพจากกระจกเงา | ระหว่างทาง |
| ว 2.3 ม.3/15 | อธิบายการหักเหของแสงเมื่อผ่านตัวกลางโปร่งใสที่แตกต่างกัน และอธิบายการกระจายแสงของแสงขาวเมื่อผ่านปริซึม | ระหว่างทาง |
| ว 2.3 ม.3/16 | เขียนแผนภาพการเคลื่อนที่ของแสง แสดงการเกิดภาพจากเลนส์บาง | ระหว่างทาง |
| ว 2.3 ม.3/17 | อธิบายปรากฏการณ์ที่เกี่ยวกับแสง และการทำงานของทัศนอุปกรณ์จากข้อมูลที่รวบรวมได้ | ปลายทาง |
| ว 2.3 ม.3/18 | เขียนแผนภาพการเคลื่อนที่ของแสง แสดงการเกิดภาพของทัศนอุปกรณ์และเลนส์ตา | ระหว่างทาง |
| ว 2.3 ม.3/19 | อธิบายผลของความสว่างที่มีต่อดวงตาจากข้อมูลที่ได้จากการสืบค้น | ระหว่างทาง |
| ว 2.3 ม.3/20 | วัดความสว่างของแสงโดยใช้อุปกรณ์วัดความสว่างของแสง | ระหว่างทาง |
| ว 2.3 ม.3/21 | ตระหนักในคุณค่าของความรู้เรื่องความสว่างของแสงที่มีต่อดวงตา โดยวิเคราะห์สถานการณ์ปัญหาและเสนอแนะการจัดความสว่างให้เหมาะสม | ปลายทาง |

## สาระที่ 3 | วิทยาศาสตร์โลกและอวกาศ

### มาตรฐาน ว 3.1 — ระบบสุริยะและเทคโนโลยีอวกาศ
> เข้าใจองค์ประกอบ ลักษณะ กระบวนการเกิด และวิวัฒนาการของเอกภพ กาแล็กซี ดาวฤกษ์ และระบบสุริยะ รวมทั้งปฏิสัมพันธ์ภายในระบบสุริยะที่ส่งผลต่อสิ่งมีชีวิต

| รหัส | ตัวชี้วัด | ประเภท |
|------|-----------|--------|
| ว 3.1 ม.3/1 | อธิบายการโคจรของดาวเคราะห์รอบดวงอาทิตย์ด้วยแรงโน้มถ่วง จากสมการ F = Gm₁m₂/r² | ระหว่างทาง |
| ว 3.1 ม.3/2 | สร้างแบบจำลองที่อธิบายการเกิดฤดู และการเคลื่อนที่ปรากฏของดวงอาทิตย์ | ปลายทาง |
| ว 3.1 ม.3/3 | สร้างแบบจำลองที่อธิบายการเกิดข้างขึ้นข้างแรม การเปลี่ยนแปลงเวลาการขึ้นและตกของดวงจันทร์ และการเกิดน้ำขึ้นน้ำลง | ปลายทาง |
| ว 3.1 ม.3/4 | อธิบายการใช้ประโยชน์ของเทคโนโลยีอวกาศและยกตัวอย่างความก้าวหน้าของโครงการสำรวจอวกาศ | ปลายทาง |
